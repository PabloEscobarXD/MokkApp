DESCRIPCIÓN.

La aplicación es una plataforma para la creación y gestión de recetas culinarias, diseñada para ayudar 
a los usuarios a descubrir, crear y compartir recetas de café y otros platillos. Los usuarios pueden ingresar 
detalles de sus recetas, como ingredientes, instrucciones, tipo de grano, técnica de extracción y equipo necesario. 
Además, la aplicación sugiere productos relacionados para cada receta.

Requerimientos:

- Creación de Recetas.
- Visualización de Recetas (Sistema o creados por otros usuarios).
- Gestión de Usuarios.
- Gestión de recetas preferidas.
